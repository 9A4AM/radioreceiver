What is this?

In few words: an HTML5 web-based RTL-SDR stereo FM radio receiver.

In more words: this webpage can use a cheap USB digital TV receiver to capture radio signals, do FM demodulation in the browser, and play the resulting audio signal through your computer's speakers or headphones.

This is called SDR (Software-Defined Radio) because all the radio signal processing is done by software running in the computer, not by specially built hardware.

This is not streaming radio. You must plug an RTL-SDR receiver into your computer's USB port to use this webpage.

Requirements

  • An RTL2832U digital TV receiver with an R820T or R820T2 tuner.
  • A browser that supports the HTML5 USB API.
  • Windows: use Zadig to install a WinUSB driver for the RTL2832UHIDIR/RTL2832U device.
  • Linux: you may need to disable the dvb_usb_rtl28xxu driver.
  • Mac, ChromeOS, and Android: it should work out of the box.

Plans

This webpage will be updated to support modern RTL-SDR devices.

It will also be updated to provide the type of UI that people have come to expect of RTL-SDR software, with a waterfall, multiple modulation schemes, etc.
